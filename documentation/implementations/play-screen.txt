# Play Screen Implementation Plan

## Overview
This document outlines the plan to consolidate the existing /lessons and /reserve routes into a unified "play" page that provides a comprehensive view of ways to engage with The Pickle Co's facilities and services. The /calendar page will remain unchanged, but we will reuse some of its components in the new play page.

## Component Details

### 1. AnnouncementBanner (✅ COMPLETED)
- Hidden by default
- Ready for future announcements
- Configurable through admin panel

### 2. EventSpotlight (✅ COMPLETED)
- Reused from calendar page
- Shows featured events
- Handles registration/unregistration

### 3. CoachesSection (✅ COMPLETED)
- Displays available coaches
- Booking modal integration
- Supports "any available coach" option

### 4. UserRegistrations (✅ COMPLETED)
- Shows upcoming bookings
- Combines lessons, courts, events
- Handles cancellations
